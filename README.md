## Course Management

Course management app in React.

### Features:

- Authentication
- Admin Dashboard:
  - Courses CRUD
  - Users CRUD
  - Courses Subscription Management
  - Statistics Graphs
  - Lessons and materials management
- User:
  - Personal Courses
- Profile Page

### Libraries

- Shadcn (with Tailwind)
- React Hook Form
- React Query
- Json Server
- React Router
- ... and more

### Installation

- Clone the repository
- Command ``` npm install ```
- Command ``` npm run:dev ``` to launch Json Server, the File Managemenet server and React App
